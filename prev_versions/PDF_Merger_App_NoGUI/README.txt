PDF Merger Tool - Instructions

1. Double-click pdf_reader.exe
2. Enter full folder paths when asked:
   - Source folder with PDFs
   - Attachment 1 PDF
   - Attachment 2 PDF
   - Output folder path (will be created if it doesn't exist)
3. Merged PDFs will be saved in the output folder.
4. A merge_log.txt file will be generated to show progress.
